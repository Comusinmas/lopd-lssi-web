# Adaptació LOPD-GDD i LSSI-CE – I+ Web

## 1. Anàlisi de cookies
S’ha realitzat una anàlisi de cookies mitjançant l’eina CookieYes.

Inicialment no es detectaven cookies perquè el JavaScript de gestió de consentiment no estava actiu.
Un cop implementat l’avís de cookies, el lloc web utilitza:

- Cookie tècnica pròpia: `cookies_accepted`
  - Finalitat: guardar el consentiment de l’usuari
  - Tipus: tècnica
  - Durada: 1 any

Les cookies analítiques (Google Analytics) només es carreguen si l’usuari accepta les cookies.

## 2. Accions realitzades per complir la normativa
- Afegida pàgina d’Avís Legal
- Afegida pàgina de Política de Privacitat
- Implementat avís de cookies amb acceptació explícita
- Bloqueig de Google Analytics fins a l’acceptació de cookies
- Modificat el menú per incloure enllaços legals
- Informació clara sobre responsable, finalitat i drets de l’usuari

## 3. Compliment normatiu
El lloc web compleix amb:
- LOPD-GDD
- RGPD
- LSSI-CE
